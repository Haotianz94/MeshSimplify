#MeshSimplify
A Mesh Simplification Algorithm

##Features
 * Based on Edge Collapse
 * Used Error Quadrics
 * Used C++
 * Developed on Visual Studio 2012

##Data Structures
 * Binary Heap based on Array
 * Linked List based on Array
